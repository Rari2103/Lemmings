# HOW TO USE - NO VS STUDIO NEEDED!

## Easy Launch Methods

### Method 1: Double-Click BAT Files (Easiest!)

We created simple launcher files you can double-click:

1. **START_DASHBOARD.bat** - Opens the web dashboard
   - Location: `trading_agent/START_DASHBOARD.bat`
   - Double-click to start
   - Browser opens automatically to http://localhost:5000

2. **CHECK_WALLET.bat** - Quick wallet balance check
   - Location: `trading_agent/CHECK_WALLET.bat`
   - Shows your current USDC balance
   - No browser needed

### Method 2: Create Desktop Shortcut

1. Go to: `C:\Users\ttosh\trading_agent\`
2. Right-click `START_DASHBOARD.bat`
3. Click "Send to" â†’ "Desktop (create shortcut)"
4. Rename shortcut to "AI Trading Agent"
5. Now you can launch from desktop!

### Method 3: Pin to Taskbar

1. Right-click `START_DASHBOARD.bat`
2. Click "Pin to taskbar"
3. Click taskbar icon to launch
